How to install the project:

1. Install Python:
    - Visit the Python website (https://www.python.org/) and download the latest version of Python.
    - Run the installer and follow the instructions to install Python on your system.

2. Create a Virtual Environment:
    - Open your command prompt or terminal.
    - Navigate to the directory where you want to create your Django project.
    - Run the following command to create a virtual environment: python3 -m venv myenv

3. Activate the Virtual Environment:
    - On Windows: myenv\Scripts\activate
    - On macOS/Linux: source myenv/bin/activate

4. Clone the GitHub Repository:
    - Ensure that Git is installed on your system. If not, visit the Git website (https://git-scm.com/) and download the latest version.
    - In the command prompt or terminal, navigate to the directory where you want to clone the repository.
    - Run the following command to clone the repository: git clone https://github.com/pvalls4/yeastar.git

5. Install Project Requirements:
    - Navigate to the cloned project directory: cd yeastar
    - Run the following command to install the project requirements: pip install -r requirements.txt

6. Configure the Project:
    - In the project directory, you may find a file named settings.py or a similar configuration file. Check if any specific configuration is required, such as database settings, API keys, or environment variables.
    - Make any necessary configuration changes according to your environment and requirements.
